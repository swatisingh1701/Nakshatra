# 🌌 Nakshatra

**Explore the Wonders of the Universe**

Nakshatra is a professionally designed cosmic-themed website that offers an immersive journey through space, planets, constellations, and astronomical discoveries. Built entirely using **HTML, CSS, and JavaScript**, the project combines beautiful UI design with interactive animations to create a premium frontend experience.

---

## ✨ Overview

Nakshatra brings together modern web design, smooth animations, and engaging space content to create an educational and visually appealing experience inspired by the beauty of the cosmos.

The project focuses on:

* Interactive user experience
* Advanced CSS animations
* JavaScript DOM manipulation
* Responsive design
* Clean code organization

---

## 🚀 Features

### 🌠 Custom Animated Cursor

* Glowing cursor dot with smooth follower effect
* Interactive hover animations
* Automatically disabled on mobile devices

### 🌌 Hero Section

* Multi-layer animated star background
* Rotating and glowing nebula effects
* Animated title with color transitions
* Floating moon with parallax scrolling
* Interactive "Begin Journey" button

### 🚀 About Space

* Beautiful informational cards
* Hover effects with glow animations
* Animated icons

### 🪐 Dynamic Planet Cards

Explore:

* ☀ Sun
* 🌍 Earth
* 🌕 Moon
* 🔴 Mars
* 🪐 Saturn
* ♃ Jupiter
* 🔵 Neptune

Each card contains:

* Planet image
* Distance from the Sun
* Description
* Fun facts

### ✨ Constellation Gallery

Featuring:

* Orion
* Ursa Major
* Cassiopeia
* Scorpius

With hover animations and image zoom effects.

### 📚 Space Facts Generator

Generate random facts with a single click.

Includes:

* Smooth text transitions

* Multiple pre-defined facts

### 💡 Did You Know?

* Automatically changing facts
